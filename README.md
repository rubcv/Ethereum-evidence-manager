# Diseño e implementación de un entorno de gestión forense basado en Blockchain

Para ejecutar:

* Dirigirse al directorio SRC con la terminal
* Ejecutar ./recompile.sh
      recompile.sh:
          > rm -r -d build
          > truffle compile
 * Ejecutar Ganache-UI
 * Ejecutar app.js 
          > node app.js
* Dirigirse a un navegador Chrome con Metamask instalado:
          > localhost:8888
